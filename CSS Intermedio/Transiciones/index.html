Transiciones

Las Transiciones en CSS permiten animas los cambios en las propiedades de un elemento cuando estos ocurrem, suavizando la trnasición de un estado a otro. En lugar de cambiar una propiedad de manera instantánea, la transición define cómo se debe interpolar el cambio a lo largo del tiempo

Sintaxis básica de tansition

transition es un atajo que agrupa otras propiedades relacionadas con las transiciones: transition: "property" "duration" "timing-function" "delay";

  • propertyEspecifica que valor se quiere animar (por ejemplo: width, background-color, color, etc). Tambien se puede usar all para aplicar la transición a todas las propiedades que cambien, pero es mas recomendable usarla por valor.
  • duration El tiempo que durará la transición (por ejemplo, 1s - 1 segundo.)
  • timing-function Define cómo progresa la transición a lo largo del tiempo. algunos valores comunes son:
    • ease: lento-rapido-lento
    • linear: constante
    • ease-in: lento-rapido
    • ease-out: rapido-lento
    • ease-in-out: ease-in + ease-out
    • funcion cubic-bezier: Click para ir a la pagina
  • delay Un tiempo opcional que retrasa el inicio de la transición.

Transiciones Múltiples

También es posible animar varias propiedades simultáneamente, cada una con sus propios valores de duración, retardo y función de tiempo. en este ejemplo
transition:
background-color .5s ease,
border .5s ease-in,
color .5s ease-out, border-radius
.5s ease-in-out;

Consideraciones importantes

  • No todas las propiedades CSS pueden animarse. Las propiedades animables son aquellas que tienen valores numéricos o de color, como width, height, margin, opacity, color entre otras.
  • Las transiciones no tienen efecto si el cambio de la propiedad no es visible, por ejemplo, si cambias una propiedad que no afecta al diseño o estilo visual del elemento, la transición no será apreciable.
