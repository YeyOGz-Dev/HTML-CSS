Control de flujo de texto

Las propiedades de control de flujo de texto en CSS permiten gestionar cómo se presenta y comporta el texto en un contenedor. Son especialmente útiles para controlar la alineación, el ajuste, el espaciado y el corte de texto cuando el contenido excede el área disponible.

  1. white-space: Controla cómo se manejan los espacios en blanco y las nuevas líneas en el contenido de texto.
    • normal:Se ignoran los saltos de línea múltiples, y el texto se ajusta dentro del contenedor.
    • nowrap: El texto no se ajusta, incluso si se desborda.
    • pre: Preserva los espacios en blanco y saltos de línea que le demos en el HTML, pero se deborda.
    • pre-wrap: Preserva los saltos de línea, pero el texto puede ajustarse si es necesario.
    • pre-line: Combina las características de normal y pre; preserva los saltos de línea, pero se ignoran los espacios en blanco adicionales.
  2. text-overflow: Esta propiedad controla cómo se muestra el texto cuando se desborda el contenedor asignado.
    • clip: El texto se corta bruscamente al final del contenedor sin mostrar ninfuna indicación.
    • ellipsis: Si el texto se desborda, se muestra un "..." para indicar que hay más contenido oculto.
    • string: Cualquier cadena de texto personalizada que se desee mostrar al funal cuando se corte el texto.
  3. overflow-wrap:Esta propiedad permite controlar cómo las palabras largas se dividen o ajustan dentro de un contenedor
    • normal: Las palabras no se dividirán y el contenido se desbordará si es necesario.
    • break-word: Las palabras re romperán en múltiples líneas si exceden el ancho del contenedor.
