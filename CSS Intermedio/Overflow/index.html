Overflow

El Overflow en CSS controla cómo se comporta el contenido cuando sobrepasa elárea del contenedor. Se utiliza principalmente para definir qué hacer cuando el contenido excede las dimensiones especificadas del elemento.

Propiedades de Overflow

  1. Overflow: Esta propiedad principal aplica tanto al eje horizontal como al vertical. Si necesitas controlarlo de forma independiente, puedes usar overflow-x (horizontal) y overflow-y (vertical).

    2. Valores más comunes:

    • visible (Valor por defecto). El contenido desbordado es visible fuera del contenedor.
    • hidden El contenido que excede las dimensiones del contenedor se oculta.
    • scroll Siempre muestra una barra de desplazamiento, incluso si no hay contenido adicional.
    • auto Muestra una barra de desplazamiento solo cuando es necesario, es decir, cuando el contenido sobrepasa el contenedor.
