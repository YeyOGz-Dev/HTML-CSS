Selectores

Selectores por atributo

  • Estos seleccionan elementos basados en un atributo o valor especifico de un atributo y tiene algunas variaciones adicionales por ejemplo:
    • Con esto estamos seleccionando a TODAS las etiquetas "input" que son de tipo "text" y le estamos dando su estilo. ¡Su valor tiene que ser exacto.!
      input[type="text"] { 'estilos' }
    • $= Con esto seleccionamos a los elementos cuyo valor TERMINEN con el valor especificado. En este ejemplo es como si seleccionaramos TODOS los elementos que terminen en "jpg"
      img[src$="jpg"] { 'estilos' }
    • ^= Con esto seleccionamos a los elementos cuyo valor EMPIEZE con el valor especificado. En este ejemplo es como si seleccionaramos TODOS los elementos que empiecen por "png"
      img[src$="png"] { 'estilos' }
    • *= Con esto seleccionamos a TODOS los elementos cuyo valor del atributo contiene la cadena especificada. Su valor no tiene que ser exacto, por ejemplo si la cadena en el html dice youtube.com solo basta con poner youtube y funcionara, pero algo importante es que no se deben mezclar las mayuculas y las minusculas ya que eso si daria una variacion.
      a[href*="example"] { 'estilos' }
      Ejemplo de *=

    • Selectores por valor descendente

  • Estos seleccionan elementos que son descendientes de otro, un espacio entre dos selectores indica una relacion descentente por ejemplo:
    • div p : Así estamos seleccionando TODAS las etiquetas p que estan dentro de un div, en el css seria de la siguiente manera:
      div p {'estilos'}

      p dentro de un div para el ejemplo

      Esto tambien se puede hacer en orden descendente con clases, id etc por ejemplo:
      etiqueta .clase etiqueta {'estilos'}

    Selector de Hijo Directo

  • Estos seleccionan elementos que son descendientes de otro, un espacio entre dos selectores indica una relacion descentente por ejemplo:
    • ul > .list-item : Así estamos seleccionando TODAS las etiquetas 'li' de clase 'list-item' que estan dentro de una 'ul', en el css seria de la siguiente manera:
      ul > .list-item {'estilos'}
      • .list-item dentro de una etiqueta 'ul'
