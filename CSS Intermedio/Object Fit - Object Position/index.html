Object Fit y Object Position

object-fit y object-position Son propiedades de CSS que controlan cómo se ajusta un contenido multimedia dentro de un contenedor. Estas propiedades son útiles cuando se trabaja con imágenes o videos, especialmente cuando las dimensiones del contenido no coinciden exactamente con las dimensiones del contenedor.

  1. object-fit: Especifica cómo debe ajustarse un elemento "img" o "video" dentro de su contenedor, definiendo cómo se escala y se recorta el contenido para que encaje.

      Valores principales

    • fill:(Por defecto) Hace que el contenido se ajuste al tamaño del contenedor, distorsionando la imagen si es necesario para que se rellene completamente.
    • contain: Escala la imagen para que se ajuste dentro del contenedor sin distorsión, pero dejando espacio vacío si las proporciones no coinciden.
    • cover: Escala la imagen para que cubra todo el contenedorm manteniendo las proporciones, pero recortando las partes que no encajan.
    • none: La imagen no se escala no se recorta, mantiene sus dimensiones originales.
    • scale-down: Se comporta como none o contain, lo que reduzca más la imagen (es decir, si la imagen es más pequeña que el contenedor, no se escala; si es más grande, se ajusta como contain).
  2. La propiedad object-position define la posición del contenido dentro de su contenedor cuando se usa con object-fit (generalmente con valores como contain o cover). Por ejemplo, cuando una imagen es más pequeña que el contenedor, object-position puede usarse para alinearla en diferentes partes del contenedor (centro, izquierda, derecha, etc.).
