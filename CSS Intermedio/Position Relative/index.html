Position

La propiedad position en CSS permite controlar cómo se posicionan los elementos dentro de un documento web. Hay varios valores que se pueden asignas a position, cada uno con un comportamiento diferente.

position:relative

Esta propiedad se utiliza para desplazar un elemento desde su posición original dentro del flujo del documento, sin sacarlo de dicho flujo. Los elementos con esta propiedad siguen ocupando el mismo espacio en el diseño, pero visualmente pueden moverse respecto a su posición inicial.

Características clave

  • El elemento permanece en el flujo del documento: Anque se pueda mover el elemento usando las propiedades de desplazamiento (top, right, bottom, left), el espacio que ocupa dentro de su contenedor sigue reservado.
  • Desplazamiento relativo a su posición original: Las propiedades de desplazamiento desplazan el elemento con relación a su posición inicial. No afecta a los elementos hermanos (otros elementos dentro del mismo contenedor) los cuales atuarán como si el elemento NO se hubiera movido.

Propiedades de desplazamiento:

  • top: Mueve el elemento hacia abajo.
  • right: Mueve el elemento hacia la izquierda.
  • bottom: Mueve el elemento hacia arriba.
  • left: Mueve el elemento hacia la derecha.

    • Solo se podria dar los valores top y left ya que son los valores principales de desplazamiento.

  • z-index: Define el orden en que los elementos se apilan unos sobre otros (en el eje Z). Cuanto mayor sea el valor, mas "adelante" estará el elemento en comparación con los demás.
    Es recomendable ponerle un z-index al contenedor padre si se va a ausar, y usarse con valores de 10 en 10.

Uso típico:

  • Se usan comúnmente cuando se quiere ajustar la posición de un elemento si alterar el flujo del documento o afectar otros elementos.
  • Es útil como referencia para elementos hijos con position:absolute, ya que el hijo se posicionará en relación a su contenedor padre con position:relative.
