PseudoClases Nuevas

Las nuevas PseudoClases :is(), :where(), :has() son más avanzadas y recientes en CSS, y están diseñadas para facilitar la selección y aplicación de estilos a elementos.

  • :is()

    Esta simplifica la escritura de selectores complejos, permitiéndo combinar varios selectores en uno solo sin repetir reglas de estilo.
    El selector dentro de :is() se evalúa, y si uno de los selectores coincide, se aplican los estilos.
    En mis propias palabras es que, si tenemos un div y dentro de este queremos seleccionar varios p de clase p1,p2,p3 podemos darle el mismo estilo a cada uno.
    Esto mas que todo sirve por si tenemos varios divs y dentro de cada uno tenemos varios p, podremos seleccionar un solo div y seleccionar los elementos que queremos estilizar sin tener que volver a llamar al div que seleccionamos por ejemplo:

  • :where()

    Este es igual que el :is() La unica diferencia es que este no afecta el cálculo de la especificidad para ese selector.
    Es util para cuando se desea aplicar estilos de "bajo peso" que puedan se fácilmente sobreescritos por otras reglas.

  • :has()

    Esta es la mejor y la mas poderosa porque permite seleccionar un elemento basado en la presencia de un descendiente o contenido específico. en otras palabras, es una PseudoClase padre, algo que no era posible antes en CSS.
    Es mis propias palabras, esto sirve para decir como, si un elemento padre TIENE como hijo por ejemplo un p, que se le den estilos, si no tiene el hijo p, entonces no tendra los estilos
