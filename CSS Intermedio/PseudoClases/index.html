PseudoClases

Las PseudoClases en CSS son palabras clave que se añaden a un selector para especificar un estado especial del elemento seleccionado. Permiten aplicar estilos basados en el estado de un elemento sin necesidad de modificar el HTML.

PseudoClases mas comunes:

  1. :hover

    Aplica estilo cuando el cursor pasa por encima del elemento.
    Pasa el mouse por encima de mi

  2. :active

    Se activa mientras el usuario este clicando el elemento.
    Click Aqui!

  3. :focus

    Aplica estilo cuando un elemento esta enfocado como como los inputs

  4. :visited

    Aplica estilos a los enlaces que ya han sido visitados.
    Enlace Visitado Enlace NO Visitado

  5. :link

    Aplica estilos a los enlaces que aun NO han sido visitados.
    Enlace Visitado Enlace NO Visitado

  6. :first-child

    Selecciona el primer hijo de un elemento, por ejemplo si tenemos una lista y queremos selccionar el primer list-item (li).

  7. :last-child

    Selecciona el ultimo hijo de un elemento, por ejemplo si tenemos una lista y queremos selccionar el ultimo list-item (li).

  8. :nth-child(#)

    Selecciona el elemento hijo dandole el #ro de posicion, por ejemplo si tenemos una ul y esta tiene 5 li, usamos ( nth-child(2) ) para seleccionar el segundo elemento.

  9. :not()

    Aplica el estilo a los elementos que no coinciden con su selector (los excluye). Esto sirve para cuando por ejemplo tenemos una lista de items y queremos seleccionar los items faltantes segun la clase.

  10. :disabled

    Selecciona y aplica estilos a los elementos que queremos deshabilitar. Este se le debe aplicar directamente al input y luego en el css llamar al input + :disabled

  11. :checked

    Aplica estilos a los elementos que están seleccionados o marcados, como checkboxes o radio buttons
