position:absolute

La propiedad position:absolute en CSS se utiliza para colocar un elemento en una posición específica dentro de su contenedor. Un elemento con esta propiedad se posiciona en relación al contenedor posicionado más cercano (es decir, un contenedor que tenga una propiedad de position diferente a la que viene por defecto que es static.)

Características clave

  • Posicionamiento fuera del flujo normal:

    Los elementos con position:absolute se eliminan del flujo normal del documento web, lo que significa que no ocupan espacio en la disposición del documento. Otros elementos actúan como si el elemento absoluto no estuviera allí.

  • Referencia a un contenedor posicionado:

    Si no encuentra un ancestro con un valor de position especifico, se posiciona en relación a la ventana del navegador (Osea a la etiqueta html).

  • Porpiedades de posicionamiento:

    Se pueden usar las propiedades top, right, bottom y left para especificar las coordenadas exactas. Estas propiedades definen la distancia del elemento con respecto a los bordes de su contenedor posicionado.
