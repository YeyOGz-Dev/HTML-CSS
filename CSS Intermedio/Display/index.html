Display

En CSS, la propiedad display define cómo un elemento HTML debe comportarse en el flujo del documento y cómo se posicionan y distribuyen sus contenidos. Dependiendo del valor asignado a un elemento puede comportarse como un bloque, una fila en línea, un contenedor flexible, una cuadrícula, entre otras...

Valores comunes de display y su comportamiento.

  • display:block

    Los elementos con display:block ocupan todo el ancho posible, se apila uno debajo del otro, comenzando en una nueva línea. Hay etiquetas que vienen con este valor por defecto.
    Soy un bloque
    Otro bloque

  • display:inline

    Un elemento con display:inline no comienza en una nueva línea y solo ocupa el ancho necesario, es decir, el contenido. Elementos como span o a son inline por defecto. Los elementos se muestran en la misma línea, uno junto al otro. A estos no se les puede dar dimensiones como margin ni padding ya que ocupan literalmente lo que miden.
    Soy inline Soy inline

  • display:inline-block

    Este combina los aspectos de block e inline. Se comporta como un elemento en línea en cuanto a su disposición en la página, pero permite especificar dimensiones como a los elementos de bloque. Esta es la mejor manera para utilizar.

  • display:none

    Elimina el elemento del flujo del documento; no se renderiza y no ocupa espacio, literalmente es como si borraramos el elemento. A diferencia de visibility:hidden que este hace que el elemento desaparezca pero si ocupa el espacio.
