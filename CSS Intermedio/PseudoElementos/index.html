PseudoElementos

Los PseudoElementos son palabras clave que se añaden a selectores para aplicar estilos a partes específicas del contenido de un elemento.

Principales PseudoElementos

  • ::before

    Este inserta contenido ANTES del contenido de un elemento, Se utiliza para agregar elementos decorativos, como iconos o texto adicional por ejemplo:

    Parrafo before

  • ::after

    Este inserta contenido DESPUES del contenido de un elemento, Se utiliza para agregar elementos decorativos, como iconos o texto adicional por ejemplo:

    Parrafo after

  • ::first-letter

    Aplica estilos a la primera letra del elemento.

    Parrafo first-letter

  • ::first-line

    Aplica estilos a la primera línea del texto dentro de un elemento, Es útil para dar formato especial a los encabezados de párrafos largos.

    Parrafo first-line Lorem ipsum dolor sit, amet consectetur adipisicing elit. Quibusdam, harum?

  • ::selection

    Estiliza la parte del texto que se selecciona con el cursor.

    Selecciona este texto

  • ::placeholder

    Aplica estilos al placeholder de un input o de un textarea.

  • ::marker

    Este se usa para aplicar estilos a los marcadores de las listas li. Permite modificar las viñetas o numeración predeterminada de las listas.
