CSS FLEXBOX

FLEXBOX es un sistema de diseño en CSSque permite la creación de layouts flexibles y adaptativos para distribuir elementos dentro de un contenedor. Está diseñado para distribuir espacio dentro de un contenedor y manejar la alineación , el tamaño y la distribución de los elementos, incluso cuando sus tamaños no son fijos. Es especialmente útil para crear diseños responsivos.

Conceptos clave de FLEXBOX

  1. Contenedor flexible (Flex container):

    Es el elemento padre que contiene uno o más elementos flexibles. Se define como un contenedor flexible al aplicar la propiedad display:flex o display:inline-flex. Cuando un contenedor es flexible, todos sus hijos directos se convierten en elementos flexibles (flex-items)

  2. Ejes en Flexbox:

    Flexbox trabaja en dos ejes principales: el eje principal (main axis) y el eje cruzado (cross axis).

    • Main axis

      Es el eje en el que se distribuyen los elementos de manera predeterminada. Por defecto, es horizontal, pero puede cambiar dependiendo de la propiedad flex-direction

    • Cross axis

      Es perpendicular al eje principal. Si el eje principal es horizontal, el eje cruzado será vertical, y viceversa.

  3. Propiedades del contenedor Flexbox

    Estas propiedades afectan a cómo se comportan los elementos dentro del contenedor flexible.

    • flex-direction
    • flex-wrap
    • justify-content
    • align-items
    • align-content
  4. Propiedades de los elementos flexibles

    Estas propiedades controlan el comportamiento de los elementos individuales dentro del contenedor flexible

    • flex-grow
    • flex-shrink
    • flex-basis
    • align-self
  5. Orden y alineación de los elementos

    Flexbox permite cambiar el orden visual de los elementos y alinearlos de maneras que no serían posibles con otros métodos de diseño.

    • order
    • align-self

Flex Direction

flex-direction es una propiedad fundamental de Flexbox en CSS. Su función principal es definir la dirección en la que los elementos hijos de un contenedor flexible (flex-container) se alinean dentro del mismo. El eje principal sobre el cual se organizan estos elementos es lo que cambia con esta propiedad.

Valores de flex-direction:

  1. row (por defecto):
    • Los elementos se organizan en una fila, alineados horizontalmente de izquierda a derecha (LTR).
    • El eje principal es horizontal y El eje transversal es vertical.
  2. row-reverse:
    • Similar a row, pero los elementos se alinean de derecha a izquierda.
    • El eje principal es horizontal pero la dirección cambia.
  3. column:
    • Los elementos se organizan en una columna, alineados verticalmente de arriba hacia abajo.
    • El eje principal es vertical, y el eje transversal es horizontal.
  4. column-reverse:
    • Similar a column, pero los elementos se alinean de abajo hacia arriba.
    • El eje principal sigue siendo vertical, pero en dirección inversa.

Flex Wrap

flex-wrap es una propiedad de flexbox que controla si los elementos hijos dentro de un contenedor flexible deben mantenerse en una sola línea o pueden "envolverse" (hacer wrap) en varias líneas cuando no hay suficiente espacio en el contenedor.
Por defecto, los elementos en un contenedor flexible se colocan en una sola línea, incluso si esto significa que se reducirán o se comprimirán para ajustarse al tamaño del contenedor. flex-wrap permite cambiar este comportamiento.

Valores de flex-wrap

  1. nowrap (por defecto):
    • Los elementos se mantienen en una sola línea, sin importar si hay suficiente espacio disponible en el contenedor.
    • Los elementos pueden reducirse en tamaño o sobrepasar el contenedor si es necesario.
  2. wrap
    • Los elementos puede "envolverse" a una nueva lína si no hay suficiente espacio en la línea actual.
    • Los elementos adicionales se colocan en una nueva línea (o columna si la dirección es vertical).
  3. wrap-reverse
    • Similar a wrap, pero los elementos se envuelven en la dirección inversa.
    • En lugar de colocar las nuevas líneas debajo de la primera, se colocan por encima (en el caso de una disposición en filas), o a la izquierda (en el caso de una disposición en columnas).

Justify Content

Para controlar la alineción y distribucion a lo largo del eje principal, se utiliza la propiedad justify-content. Esta propiedad afecta cómo los elementos dentro del contenedor se distribuyen horizontal o verticalmente, dependiendo de la dirección del contenedor.

Valores de justify-content

  1. start (por defecto):

    Alinea los elementos al principio del eje principal.

  2. end:

    Alinea los elementos al final del eje principal.

  3. center:

    Los elementos se centran a lo largo del eje principal.

  4. space-between:

    Distribuye los elementos uniformemente con el primer elemento alineado al principio y el último alineado al final.

  5. space-around:

    Los elementos tienen un espacio igual alrededor de ellos (el espacio entre los elementos es más grande que el espacio de en los extremos).

  6. space-evenly:

    Distribuye los elementos con espacios iguales entre ellos y en los extremos.

Align Items

Alinea los elementos a lo largo del eje transversal, que es perpendicular al eje principal. Afecta al posicionamiento de los elementos en una sola línea flexible.

Valores de align-items

  1. stretch (por defecto):

    Los elementos se estiran para llenar el contenedor a lo largo del eje transversal (si no tienen un tamaño fijo).

  2. start:

    Alinea los elementos al principio del eje transversal.

  3. end:

    Alinea los elementos al final del eje transversal.

  4. center:

    Los elementos se centran a lo largo del eje transversal.

  5. baseline:

    Alinea los elementos a lo largo de sus líneas de base tipográfica (donde comienza el texto).

Align Content

Esta propiedad se utiliza cuando los elementos están envueltos en varias líneas (cuando flex-wrap esta activado). Basicamente es lo mismo que el align-items solo que esta propiedad se utiliza cuando hay mas de 1 linea de elementos hijos. Controla cómo se distribuyen las líneas dentro del contenedor.

Valores de align-content

  1. start:

    Las líneas se alinean al principio del eje transversal.

  2. end:

    Las líneas se alinean al final del eje transversal.

  3. center:

    Las líneas se alinean a lo largo del eje transversal.

  4. space-between:

    Las líneas se distribuyen uniformemente con la primera línea al principio y la última al final.

  5. space-around:

    Las líneas tienen un espacio igual alrededor de ellas.

  6. space-evenly:

    Las líneas están igualmente espaciadas, incluyendo los extremos.

  7. stretch:

    Las líneas se estiran para llenar todo el contenedor.

Flex Grow

La propiedad flex-grow define la capacidad de un elemento para crecer y ocupar el espacio disponible en el contenedor, en proporción a los demás elementos.

  • Valor:

    un número positivo, generalmente un número entero o decimal

  • Por defecto: 0

    El elemento no crece.

    • Si un elemento tiene un valor de flex-grow mayor que los otros, ocupará más espacio del contenedor.

Flex Shrink

La propiedad flex-shrink controla la capacidad de un elemento para encogerse si el tamaño del contenedor es más pequeño que la suma de los tamaños de todos lo elementos.

  • Valor:

    un número positivo (o 0 para que no se encoja), generalmente un número entero o decimal

  • Por defecto: 1

    El elemento se encoje si es necesario.

    • si varios elementos tienen diferentes valores de flex-shrink, aquellos con un valor más alto se encogerán más en proporción cuando el espacio sea limitado.

En este ejemplo el div 1 tendra shrink:1, el div 2 tendra shrink:2 y asi sucesivamente.

Flex Basis

La propiedad flex-basis especifica el tamaño base de un elemento antes de que se aplique flex-grow o flex-shrink. Define el tamaño inicial del elemento a lo largo del eje principal antes de considerar cómo crecer o encogerse según el espacio disponible.

  • Valor:

    Una longitud (en px, em, rem, %, etc) o auto que es el valor por defecto.

  • Si se define flex-basis, este valor tendrá prioridad sobre width o heigth a lo largo del eje principal.

En este ejemplo el div 1 tendra shrink:1, el div 2 tendra shrink:2 y asi sucesivamente.
