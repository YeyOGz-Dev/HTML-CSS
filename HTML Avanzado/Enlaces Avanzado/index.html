Enlaces Avanzado

  • alternate: Indica que el documento enlazado es una versión alternativa del documento actual en otro idioma o formato.
  • author: Indica que el documento enlazado es la página web del autor del documento actual.
  • bookmark: Indica que el documento enlazado es un marcador a una sección específica del documento actual.
  • canonical: Indica que el documento enlazado es la versión principal del documento actual.
  • help: Indica que el documento enlazado contiene información de ayuda sobre el documento actual.
  • license: Indica que el documento enlazado contiene la licencia del software o contenido del documento actual.
  • next: Indica que el documento enlazado es la siguiente página en una secuencia de páginas.
  • nofollow: Indica que el navegador no debe seguir el enlace y no debe pasar PageRank al documento enlazado.
  • noopener: Indica que el documento enlazado debe abrirse en una nueva ventana o pestaña sin compartir las cookies del documento actual.
  • noreferrer: Indica que el documento enlazado no debe tener acceso al encabezado Referer del documento actual.
  • prev: Indica que el documento enlazado es la página anterior en una secuencia de páginas.
  • stylesheet: Indica que el documento enlazado es una hoja de estilos que se aplica al documento actual.
  • tag: Indica que el documento enlazado es una página que contiene información sobre una palabra clave o tema específico.
