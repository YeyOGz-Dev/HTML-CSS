Tablas

Las tablas en HTML se utilizan para organizar datos en filas y columnas. Son una herramienta fundamental para estructurar información de forma clara y legible.

Estructura de las tablas:

  • table: Etiqueta para crear tablas.
  • thead: Encabezado de la tabla
  • tbody: Cuerpo de la tabla
  • tfoot: Pie de la tabla. Por ejemplo el total.
  • th (table head): Define el encabezado de una fila
  • tr (table row): Define una fila de la tabla
  • td (table data): Define el dato dentro de una fila
