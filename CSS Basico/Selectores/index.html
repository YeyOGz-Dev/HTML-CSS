Formas de agregar los estilos a la pagina

Existen 3 formas:

  1. Estilos en linea: NO SE RECOMIENDA. Se le agrega el atributo "style" a la etiqueta HTML y se le dan las propiedades.
  2. Etiqueta "style" en el mismo HTML: NO SE RECOMIENDA. Con la etiqueta style tambien se pueden agregar los estilos a las etiquetas.
  3. Hoja externa: esta es la mejor practica para implementar los estilos a la pagina web, ya que es mas ordenado el codigo por que se separa totalmente el css y el HTML para cuando necesitemos escribir y leer el codigo sea mas amigable.
    Se crea una carpeta llamada CSS o styles en la carpeta padre para saber sobre que es la carpeta y luego crear un archivo llamado styles.css o index.css (esto dependera de como querramos diferenciar los archivos, ya que lo importante es que el archivo termine con .css)

Estructura para agregar los estilos en el archivo .css:

Vamos a tomar como ejemplo las etiqueta 'p'

  • Selector por elemento: Agregamos la etiqueta html y le damos sus estilos en el archivo .css: p{ propiedad = valor; }; (De esta manera se le dara el mismo estilo a TODAS las etiquetas de este tipo.)
  • Selector de clase: a la etiqueta html se le agrega el atributo 'class' y se le asigna un nombre de referencia para ese elemento (a cada elemento se le pueden agregar las clases que sean) y en el archivo .css lo haremos asi: (punto al inicio y luego el nombre de la clase de referencia) .li{ propiedad = valor; };
  • Selector de id: A la etiqueta html se le agrega el atributo id con el nombre del identificador (es importante saber que un id solo se le puede agregar a un solo elemento, no se puede tener dos o mas elementos con el mismo id0). Igual que con el selector de clase pero esta vez, como es con el selector de id es con el signo '#' (# al inicio y luego el nombre del identificador) #li{ propiedad = valor; };
