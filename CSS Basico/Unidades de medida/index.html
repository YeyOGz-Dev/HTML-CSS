Unidades de medida

las unidades de medida son esenciales para definir dimensiones, márgenes, paddings y otras propiedades de los elementos. Existen dos tipos principales de unidades: absolutas y relativas. lo mas recomendable es usar las relativas ya que son mas adaptables.

Unidades de medida ABSOLUTAS

  • Pixeles (px): La unidad más común, equivalente a un punto en la pantalla. Son unidades fijas e independientes del tamaño de la fuente o del dispositivo. Algunos de sus usos son para el posicionamiento de elementos, tamaño de imagenes, bordes, etc.
  • Centimetros, milimetros y pulgadas (cm, mm, in): Son unidades absolutas pero no se usan casi.
  • Puntos y picas (pt, pc): No se utilizan casi.

Unidades de medida RELATTIVAS

  • em: Equivalente al tamaño de la fuente actual del elemento. Permite crear diseños adaptables al tamaño de la fuente. Se usa mas que todo para los titulos, parrafos y/o menus.
  • rem: esta medida es muy similar al em, pero se basa en el tamaño de la fuente del elemento raiz (html). Ideal para diseños adaptables a diferentes dispositivos. Esta es la opcion que personalmente me gusta mas, ya que es muy facil de aplicar y se puede entender facilmente.
    En la hoja de estilos, debemos de asignarle a la etiqueta 'html' un font-size de 62.5% para que por defecto se entienda que seran 10px, y luego le asignamos el font-size al body con el valor de 1.6rem (esto significa 16px) y por ejemplo para un parrafo que queremos que tenga 30px simplemente le asignamos 3rem (Muestra real en el css.)
  • Vw (viewport width): representa el ancho del area visible del navegador
  • Vh (viewport height): Representa la altura del area visible del navegador.
  • Vmin: Es el valor menor entre vw y vh.
  • Vmax: Es el valor mayor entre vw y vh.

    • Consejos para elegir la unidad de medida adecuada:

    • Para diseños fijos:Se usan unidades absolutas (px)
    • Para diseños adaptables al tamaño de la fuente:em o rem
    • Para diseños adaptables a diferentes dispositivos:vw, vh, vmin, vmax
