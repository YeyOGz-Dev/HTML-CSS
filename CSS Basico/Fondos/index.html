Fondos

Propiedades:

  • background-color: Para darle el color
  • background-image: Para poner una imagen de fondo.
  • background-position: Define la posicion de la imagen.
  • background-repeat: Controla como se repite la imagen de fondo si es mas pequeña que el elemento

    • Se puede usar la propiedad 'background' como shorthand para estas cuatro propiedades por ejemplo:
    background{#000 url("imagen") center no-repeat}

    Otras propiedades del background:

    • background-size: Ajusta la imagen al ancho y alto, la mejor opcion es "cover" ya que este acomoda la imagen lo mejor posible, pero tambien se puede usar top, left, rigth o bottom para orientar la posicion de la imagen.
    • background-attachment: Define si la imagen de fondo se desplaza con el contenido del elemento o se queda fija en su lugar. Las opciones comunes son "scroll" (se desplaza con el contenido) y "fixed" (se queda fija). Ejemplo aqui!

Gradientes

  • Gradientes lineales: (linear-gradient) Estos gradientes transicionan entre colores en una línea recta. Puedes definir la dirección del gradiente, así como los colores y las posiciones de los puntos de parada a lo largo de la línea. Por ejemplo:
    background: linear-gradient(to bottom, #000 ,#06a);

    • Se puede elgir la direccion hacia la que va el gradiente, por ejemplo si se pone "to bottom" el primer color se desvanece desde arriba hacia abajo hasta combinarse con el otro color asignado. Tambien se le puede poner el % a cada color de cuanto queremos que se vea para ser mas especificos y se le pueden asignar mas de 2 colores.
