Box Model (Modelo de caja) IMPORTANTISIMO!

Propiedades del box model

  1. Content (contenido): El área central que muestra el contenido real del elemento, como texto, imágenes o videos.
  2. Padding (relleno): Un área que agrega espacio alrededor del contenido.
  3. Border (borde): La línea decorativa que rodea el relleno y el contenido.
  4. Margin (margen): Un área transparente que rodea todo el cuadro, incluyendo el borde y el relleno. la etiqueta "body" tiene un margin por defecto. Es importante saber que el margin no hace parte de la caja en general ya que este se agrega es AFUERA de la caja, este sirve para separar los elementos de otros.

    5. El ancho (width) y alto (height) totales de un elemento vienen determinados por el tamaño del contenido más el padding, el border y el margin. (content-box)

Osea que si hay una caja con 20px de margin arriba y abajo y debajo de esa caja hay otra con 30px de margin arriba y abajo, hace que la de abajo que tiene mayor valor valga como el margen, basicamente no se suma el margen de abajo y de arriba de cada elemento si no que se superpone y le da el valor del margen mayor.

Border

Tiene 4 aspectos fundamentales:

  • border-width: Grosor del borde
  • border-style: Estilo del borde
  • border-color: Color del borde
  • border-radius: Redondeado del borde

Es importante saber que se puede usar el shorthand (border:) para los bordes, PERO la propiedad border-radius va aparte por ejemplo:

elemento{border: 5px solid black; (width - style - color).
border-radius: 5px;
}
Muestra

Para definir un borde en uno de los lados del elemento se debe usar border-top etc ya que como a cada uno se le agregan propiedades, entonces en este caso si debe de ser separado.

El border-radius si se puede usar como shorthand, ya que podemos darle el redondeado con 1,2,3 y/o 4 valores.

Box Sizing

Define como se calcula el ancho y alto total de un elemento. el valor por defecto es box-sizing: content-box; Pero esto puede causar problemas en el diseño ya que las medidas se vuelven incalculables.
Para esto hoy en dia tenemos el box-sizing: border-box; que este nos evita estar calculando el ancho y el alto de un elemento ya que si le definimos un ancho y un alto este sera igual asi le agreguemos el padding y el borde.

  • Este es el valor por defecto. El ancho y alto solo incluyen el contenido del elemento. Los bordes y el padding se suman al tamaño final. Lo mejor es cambiarle esta opcion por border-box.
  • El ancho y alto incluyen el contenido del elemento, los bordes y el padding.

    Ventajas de usar box-sizing: border-box;

    • Facilita el diseño: Puedes definir el ancho y alto de un elemento y saber que será el tamaño total del elemento, incluyendo bordes y padding.
    • Evita problemas de diseño: No tendrás que preocuparte por que los bordes y el padding se sumen al tamaño final del elemento y causen problemas de diseño.
    • Mejora la consistencia: Todos los elementos se comportarán de la misma manera, lo que hace que tu diseño sea más consistente.
