Propiedades de texto

  • "color" sirve para darle el color al texto
  • "font-family" sirve para darle una tipografia (fuente) al texto
  • "font-size" Sirve para darle tamaño al texto, por el momento solo veremos px (pixeles.) ya que hay mas unidades de medida.
  • "font-weigth" Define el grosor del texto entre 100 y 900 donde 100 es delgado y 900 es grueso. O se pueden usar las palabras clave:
    • normal: Es el valor por defecto, equivale a 400.
    • bold: Equivale a 700.
    • bolder: Equivale al valor mas grueso que tenga la fuente.
    • lighter: Equivale al valor mas delgado que tenga la fuente.
  • "font-style" Sirve para poner la letra "cursiva", Por defecto esta en normal. Para no complicarse solo usar "italic".
  • "text-align" Se utiliza para controlar l aalineacion horizontal del texto dentro de su contenedor. Valores del text-align:

    • Lorem ipsum dolor sit amet consectetur, adipisicing elit. Iure, autem!
    • Lorem ipsum dolor sit amet consectetur, adipisicing elit. Iure, autem!

    • Lorem ipsum dolor sit amet consectetur, adipisicing elit. Iure, autem!
  • "text-decoration" Sirve para subrayar el texto, valores del text-decoration:
    • none: Valor por defecto, aunque por ejemplo la etiqueta 'a' que es un link viene subrayado, si le aplicamos el none a esa etiqueta le quitamos la linea con la que viene esa etiqueta por defecto.
    • underline: Esta subraya el texto por debajo.
    • overline: Esta subraya el texto por encima.
    • line-through: Esta subraya el texto en medio

      • Se le pueden cambiar los estilos por defecto con los que viene el "subrayado" de la siguiente manera:

      • color: Para cambiarle el color al subrayado
      • style: Para cambiarle el estilo al subrayado.
  • "line-height" Controla la altura de una linea de texto, por defecto esta en 1 (100% - normal) si por ejemplo le ponemos 3.5 seria (350% - alto). en resumen, es la separacion entre lineas verticalmente para que el texto sea mas "legible". Si se le ponen valores menores a 1, sera muy poco legible.
  • "letter-spacing" Ajusta el espacio entre las letras del texto.
  • "word-spacing" Ajusta el espacio entre las palabras del texto.
  • "text-transform" Permite convertir el texto a mayúsculas, minúsculas o solo la primera letra de cada palabra en mayúscula. Valores del text-transform:
    • uppercase: Lorem ipsum dolor sit amet consectetur adipisicing elit. Obcaecati, eius!
    • lowercase: Lorem ipsum dolor sit amet consectetur adipisicing elit. Obcaecati, eius!
    • capitalize: Lorem ipsum dolor sit, amet consectetur adipisicing elit. Adipisci, id.
